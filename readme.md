# Nechifor Blog

My custom blog generator. [See it live.][live]

![Nechifor Blog screenshot.](screenshot.png)

## Usage

Clone this repo. Build and run it by typing:

    npm run up

Go to [localhost:3000](http://localhost:3000) to see it. See other [supported
commands][sc].
